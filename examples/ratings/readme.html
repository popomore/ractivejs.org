Creating a simple star rating widget is the kind of thing that quickly turns to spaghetti if you try and build it by hand, or even with jQuery – lots of manually adding and removing classes, probably using tricks like $.index() or this.getAttribute( 'data-index' ) and other such horrors.

Because Ractive.js is stateful, and uses dependency tracking to propagate changes efficiently (in other words, it enables what is often referred to as 'reactive programming'), there are no such headaches.

This demonstration was inspired by this Quora post from Pete Hunt, lead developer of React, which sought to compare React to AngularJS.

There's way too much of this kind of Angular v Ember v React v [insert framework here] chest-beating in our community, and I don't want to add to it (though Pete's post was a legitimate defence of React). But look how much simpler the Ractive.js code is!
